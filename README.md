# Projeto de PAM2 com SQLite e Typescript

Crud de carros utilizando a tabela:
    ID_CARRO INTEGER PRIMARY KEY AUTOINCREMENT,
    NOME VARCHAR(100),
    MARCA VARCHAR(50),
    ANO INTEGER,
    COR VARCHAR(30),
    PRECO REAL,
    KM_RODADOS INTEGER

Para a Navegação foi usada a biblioteca ReactNavigation/BottonTabs
